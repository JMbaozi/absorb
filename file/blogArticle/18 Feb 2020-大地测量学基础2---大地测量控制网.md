<h3 id="1大地控制网的布网原则和方法">1.大地控制网的布网原则和方法</h3>
<h4 id="1原则">（1）原则</h4>
<ul>
<li>分级布网、逐级控制</li>
<li>保持必要的精度</li>
<li>应有一定的密度</li>
<li>应有统一的规格</li>
</ul>
<h4 id="2方法">（2）方法</h4>
<ul>
<li>三角测量法</li>
<li>精密导线测量</li>
<li>三边测量</li>
<li>边角同测法</li>
</ul>
<h3 id="2国家gps的四大网名称及其作用">2.国家GPS的四大网名称及其作用</h3>
<h4 id="1四大网名称">（1）四大网名称</h4>
<ul>
<li>国家 GPS A、B级网</li>
<li>全国 GPS 一、二级网</li>
<li>攀登项目 GPS 网</li>
<li>中国地壳运动观测网络</li>
</ul>
<h4 id="2作用">（2）作用</h4>
<ul>
<li>监测研究地壳形变与块体运动</li>
<li>检核和加强各地区天文大地网，建立统一的高精度大地基准</li>
<li>建立地心参考系，精确确定参心坐标系与地心坐标系之间的转换参数</li>
<li>精化大地水准面等等</li>
<li>同时这些全国网的建立与复测都可以成为建立我国新一代高精度地心参考系、监测和研究地壳运动的基础。</li>
</ul>
<h3 id="3工程控制网的布设一般步骤">3.工程控制网的布设一般步骤</h3>
<ul>
<li>收集资料
    <ul>
<li>地形图、大地测量成果资料（三角测量、导线测量、GPS测量、水准测量的点之记、成果表、技术总结、控制网略图及水准路线图，原有点位的觇标、标石的保存情况等 ）交通图、气象资料</li>
</ul>
</li>
<li>实地踏勘
    <ul>
<li>现状和已有点位情况</li>
</ul>
</li>
<li>图上设计
    <ul>
<li>拟定控制点的点位和网形，方法：
        <ul>
<li>把测区中、小比例尺地形图拼接起来，在其上绘出测区的范围，标明已知的高等级控制点的位置。</li>
<li>选定待测控制点并连接成一定的网形。注意：
            <ul>
<li>旧有控制点的位置适宜，应尽量与旧点重合。从已知点开始逐点向外扩展，直至选取的控制点均匀布满整个测区为止。</li>
<li>相邻点间连接而成的网形结构应合乎规范相应等级的要求。</li>
<li>网形：三角网、三边网、导线网、GPS网。</li>
</ul>
</li>
<li>点位要求
            <ul>
<li>三角网、三边网控制点间要相互通视，导线网只要求相邻点间相互通视，GPS网点间不要求相互通视，但对于使用全站仪进行进一步加密的导线测量，为保证导线测量有起算的方位角，GPS网布设时应注意导线两端位置有互相通视的两个点。</li>
</ul>
</li>
<li>进行点位、边长、方位角的精度估算
            <ul>
<li>最重要的是最弱点点位位置和最弱边边长的精度估算。</li>
</ul>
</li>
<li>复查
            <ul>
<li>检查的项目有设计的点位数量、质量是否合乎规范的要求；控制点之间是否通视；布设的项目是否齐全等。</li>
</ul>
</li>
</ul>
</li>
</ul>
</li>
<li>写出控制网技术设计书
    <ul>
<li>设计书的内容包括：测量任务，测区范围测区的自然地理、气象、交通情况，作业依据，旧有资料及其利用情况，新网的起算数据、布设等级、点的数量和网形结构、精度估算结果。对于三角网则应有觇标高度和类型设计、仪器装备和各项器材的计划数量、观测与数据处理方案和最终提供的成果。</li>
</ul>
</li>
<li>上交资料
    <ul>
<li>应提交控制网设计图纸以及控制网技术设计说明书。</li>
</ul>
</li>
</ul>
<h3 id="4控制网精度估算过程">4.控制网精度估算过程</h3>
<blockquote>
<p>无论是GPS网、导线网，在设计出具体的网形之后，都应进行点位的精度估算。<strong>平面网要估算出设计网点的点位精度，还要进行边长、方位角、相对点位的精度估算，尤其是最弱边边长、方位角和最弱点点位的精度估算。</strong></p>
</blockquote>
<p>以导线网为例，精度估算主要步骤如下：</p>
<ul>
<li>计算各观测值（边长、角度或方向）的误差方程式的系数。</li>
<li>确定各观测值的权。</li>
<li>计算未知数的权倒数。</li>
<li>计算未知数的中误差及其误差椭圆元素。</li>
</ul>
<h3 id="5等权代替法的实质">5.等权代替法的实质</h3>
<p>等权代替法的实质是将复杂的水准网通过路线合并与路线连接，简化成一条虚拟的等权路线，以便按单一路线计算最弱点高程中误差。</p>
<blockquote>
<p>设三条路线长分别为L1、L2、L3，则</p>
</blockquote>
<ul>
<li>三个高差观测值权分别为 P1=1/L1、P2=1/L2、P3=1/L3</li>
<li>结点的权 P=P1+P2+P3</li>
<li>相应的等权路线长度为L=1/P</li>
<li>结点的高程中误差为
<script type="math/tex">M_{H}=M_{W} \sqrt{\frac{1}{P}}</script>
<blockquote>
<p>Mw为每千米高差全中误差</p>
</blockquote>
</li>
</ul>
<h3 id="6gps网设计的基准要求">6.GPS网设计的基准要求</h3>
<ul>
<li>联测测区附近高等级的具有WGS-84坐标的控制点，通过GPS观测数据处理，GPS点能够得到高精度的WGS-84坐标。</li>
<li>联测一定数量高等级的国家或地方坐标系的控制点，联测一定数量的水准高程点。</li>
<li>联测点三个以上，且分布均匀。</li>
</ul>
